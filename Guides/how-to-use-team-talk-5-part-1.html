How to use Team Talk 5

By Harper

How to use Team Talk?

Team Talk is a client meant for comprehensive chats and conversations. It is used in a variety of ways within the blind community.

Note: Team Talk is made by bearware.dk and is in no way affiliated with Harper and Jakob's projects. This is only a user guide. For the official documentation, please visit the Team Talk documentation.

Downloading the Client

To download the client, visit here and select your version. The .exe file is for Windows, while iOS and Android have App Store or Google Play links. Information regarding Linux installation will be updated as it becomes available.

Windows

  • Download the latest release of the client as a .exe file. Select the file named teamtalk5.exe; this is the most up-to-date version.
  • Open the setup file once the download completes.
  • Follow the setup prompts. If you do not plan to host a server, you only need the Client component. If you do plan to host a server (requires port forwarding), download the Client and Server package.
  • After installation, open the client. It will ask to use your computer's default language; click "Yes" unless you prefer another.
  • It will ask to add a firewall exception; use your discretion here. Finally, it will ask to enable better support for screen readersâ€”it is highly recommended to select "Yes."

Adding a Server

  • Press F2 (or Fn+F2) to bring up the "Connect to a Server" dialog.
  • Tab over to Add New Server.
  • Enter the server details: IP address, ports, username, password, and encryption settings.

Admin Controls

Windows

On Windows, administrators have extensive control over the server:

  • Accounts: Press Ctrl+Shift+L to access the account menu to add, delete, or edit user rights.
  • Kick Users: Use Ctrl+K to kick from a channel or Ctrl+Shift+K to kick from the server.
  • Ban Users: Press Ctrl+Shift+B to ban a user from the server.

iOS

  • Search for "TeamTalk" on the App Store to download.
  • Open the app and tap Add New Server Entry at the top of the screen.
  • Enter the address and port details, then click Save.

iOS offers slightly fewer controls than Windows, but is still very reliable:

  • Kick Users: Select the user in the channel and tap "Kick User."
  • Ban Users: Select the user and tap "Ban User" to remove them from the server.
